## 🚀 Featured Projects

### 📉 Customer Churn Retention Analysis
> **Tools:** Python · Pandas · Matplotlib · Seaborn · EDA

**Problem:** A telecom company was losing customers with no visibility into *who* was at risk or *why*.

**What I did:**
- 🔍 Analyzed **50,000+ customer records** — cleaned nulls, handled outliers, encoded categoricals
- 📊 Performed deep EDA across tenure, contract type, payment method & monthly charges
- 💡 Found that **month-to-month customers paying via electronic check** had a **42% churn rate** — 3× the average
- 🎯 Segmented customers into High / Medium / Low churn risk tiers using Python
- 📈 Retention strategy recommendations projected a **~18% churn reduction**

**Key Insight:**
> *Early-tenure customers on flexible contracts are the highest risk — intervening at Month 3 is the most cost-effective retention window.*

### 📊 Sales Performance Dashboard — Power BI
> **Tools:** Power BI · DAX · Power Query · Row-Level Security

**Problem:** Regional sales managers relied on stale Excel files — no real-time visibility into performance.

**What I did:**
- 🏗️ Built an end-to-end **interactive Power BI dashboard** from raw CSV data
- ⚡ Wrote **15+ DAX measures** — YTD Sales, MoM Growth %, Running Totals, Rank by Region
- 🔒 Implemented **Row-Level Security (RLS)** so each manager sees only their region
- 🔄 Automated data refresh using **Power Query** — zero manual updates needed
- 📱 Designed drill-through from summary view → individual transaction level

**Key Insight:**
> *South region underperformed by 23% — not due to demand, but delayed order fulfillment. Dashboard surfaced this in real-time.*
