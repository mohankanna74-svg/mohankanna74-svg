🚀 Featured Projects
📉 Customer Churn Retention Analysis
> *Python · Pandas · Matplotlib · Seaborn · EDA*
Problem: A telecom company was losing customers with no clear understanding of why or who was at risk.
What I did:
🔍 Analyzed 50,000+ customer records — cleaned nulls, handled outliers, encoded categoricals
📊 Performed deep Exploratory Data Analysis across tenure, contract type, payment method & charges
💡 Identified that month-to-month contract customers with electronic check payments had 42% churn rate — 3× the average
📈 Built churn risk segments using Python — high/medium/low risk tiers with actionable retention triggers
🎯 Recommendations: loyalty discounts for Month 1–6 customers + auto-pay nudges reduced projected churn by ~18%
Key Insight: Early-tenure customers on flexible contracts are the highest churn risk — intervening in Month 3 is the most cost-effective retention window.

📊 Sales Performance Dashboard — Power BI
> *Power BI · DAX · Power Query · RLS*
Problem: Regional sales managers had no unified view of performance — decisions were made from stale Excel reports.
What I did:
🏗️ Built an end-to-end interactive Power BI dashboard from raw CSV sales data
⚡ Wrote 15+ DAX measures — YTD Sales, MoM Growth %, Running Totals, Rank by Region
🔒 Implemented Row-Level Security (RLS) so each regional manager sees only their data
🔄 Automated refresh pipeline using Power Query — zero manual data entry
📱 Designed mobile-responsive layout with drill-through from summary → transaction level
Key Insight: South region underperformed by 23% — not due to demand but delayed order fulfillment. Dashboard surfaced this in real-time.
